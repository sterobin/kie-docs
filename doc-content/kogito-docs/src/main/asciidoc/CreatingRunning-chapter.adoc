= Creating and running your first application with {PRODUCT} services

As a developer of business processes and rules, you can use {PRODUCT} business automation to build  cloud-native applications that adapt to your business domain and tooling.

.Prerequisites
////
// @comment: Removed for now but retaining for use in next docs. (Stetson, 18 Feb 2020)
* One of the following IDEs is installed:
** VSCode (recommended) with the {PRODUCT} https://github.com/kiegroup/kogito-tooling/releases[VSCode Extension] installed for BPMN 2.0 and DMN modeling support
**  Eclipse with BPMN2 Modeler plugin with jBPM Runtime Extension for BPMN 2.0 modeling support
////
* JDK 11 or later is installed.
* Apache Maven 3.5.3 or later is installed.
* Docker is installed.
